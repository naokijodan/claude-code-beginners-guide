AI CLIツールとは？

😊
はじめさん
「AI CLI」って聞いたことないんですが、難しいものですか？パソコンは普段メールと表計算くらいしか使わないんですが...

AI CLIツールとは、パソコンの「ターミナル」（黒い画面）から起動し、 日本語で指示を出すとAIがファイルの作成・編集・整理・プログラミングなどを 代わりにやってくれるツールの総称です。

現在、大手3社がそれぞれ独自のAI CLIツールを提供しています。 どれも「ターミナルで日本語の指示を出す → AIが作業する」という基本は同じです。

何ができるの？

📄

ファイル作成・編集

「請求書のテンプレートを作って」「この文章を校正して」など、ファイル操作をAIに任せられます。

🌐

Webページ作成

「自己紹介ページを作って」と言うだけで、HTMLファイルを自動生成。プログラミング知識は不要です。

🔍

情報の検索・整理

大量のファイルの中から必要な情報を探したり、データを整理してまとめたりできます。

🛠

自動化・効率化

繰り返し作業を自動化するスクリプトを作ったり、面倒な作業をAIに肩代わりしてもらえます。

ChatGPTのチャットとの違い

多くの方が使ったことのあるChatGPTのチャット画面は「質問して回答をもらう」ツールです。 AI CLIツールは一歩進んで、AIが直接あなたのパソコン上で作業を実行してくれます。

  • チャットAI：「こうすればいいですよ」と教えてくれる（アドバイザー）
  • AI CLIツール：「やっておきましたよ」と実際にやってくれる（実行者）
📚 用語解説
AI（人工知能）
人間のように考え、判断し、作業ができるコンピュータプログラム。各社が独自のAIモデルを開発しています。
CLI（コマンドラインインターフェース）
マウスでクリックする代わりに、キーボードで文字（コマンド）を入力してパソコンを操作する方法。
ターミナル
CLIを使うためのアプリ。Macでは「ターミナル」、Windowsでは「PowerShell」と呼ばれます。黒い画面のアレです。

プロジェクトのフォルダに CLAUDE.md というファイルを置くと、 Claude Codeが毎回そのルールを読み込んで、あなたの好みに合わせた動作をしてくれます。

Codex CLIの便利機能

安全モードの選択

Codex CLIは起動時にAIの行動範囲を3段階で指定できます。

  • suggest（提案のみ）：コマンドを提案するだけで実行しない。最も安全。
  • auto-edit：ファイル編集は自動、コマンド実行は確認が必要。
  • full-auto：すべて自動実行。慣れてきたら使う。

Gemini CLIの便利機能

Googleサービスとの連携

Googleアカウントでログインしていれば、Google検索やGoogleドライブのファイルに自然にアクセスできます。また、Geminiの大きなコンテキストウィンドウ（100万トークン）を活かして、大量のファイルを一度に処理できるのも強みです。

MCPで機能を拡張する

😊
はじめさん
基本はわかってきました！もっと色々できるようになりたいです。拡張機能ってどうやって使うんですか？

MCP（Model Context Protocol）は、AI CLIツールに「追加の能力」を付け加える仕組みです。 スマホにアプリを追加するように、新しい機能を追加できます。 主にClaude Codeで使われますが、他のツールでも対応が進んでいます。

MCPでできること

🌐

Webブラウザ操作

Webページを開いて情報を取得したり、フォームに入力したりできます。（Playwright）

💬

他のAIとの連携

Claude CodeからGPTやGeminiにも質問して、複数のAIの意見を比較できます。

📊

Google Sheets操作

スプレッドシートの読み書きを自動化できます。

🎨

Figmaデザイン取得

Figmaのデザインデータを読み取って、コードに変換できます。

🔌
Claude Code × GPT × Gemini 連携ガイド

📚 用語解説
MCP（Model Context Protocol）
AIツールに外部サービスを接続するための共通規格。「USB」のように、さまざまなサービスを同じ方法で接続できます。
Playwright（プレイライト）
Webブラウザを自動操作するためのツール。MCPを通じてAI CLIツールから使えるようになります。
JSON（ジェイソン）
データの書き方の一種。設定ファイルによく使われます。{"名前": "太郎", "年齢": 30} のような形式です。

ナレッジ管理：Obsidianとの連携

AI CLIツールで作業した内容を記録・整理するのに、Obsidianというノートアプリが相性抜群です。 作業ログ、学んだこと、メモなどを蓄積していくことで、自分だけのナレッジベースが作れます。

なぜObsidianなのか

  • ローカル保存：データが自分のPC内にあるので安心
  • Markdown形式：AI CLIツールが出力する形式とそのまま互換性がある
  • 双方向リンク：ノート同士を繋げて、知識のネットワークが作れる
  • 無料で使える：基本機能は完全無料

AI CLI × Obsidian の活用例

1

開発ログの自動作成

AIに「今日の作業内容をObsidianノートにまとめて」と頼むと、作業ログを自動生成してくれます。

2

議事録の整理

会議メモをAIに渡して「要点を整理してObsidianに保存して」と指示できます。

3

学習ノートの蓄積

新しく学んだことをノートに残し、後からAIに「過去のノートを参照して」と依頼できます。

よくある質問・トラブル対応

😊
はじめさん
使っていて困ったことが出てきました。みんなが気になる質問をまとめてもらえますか？

Node.jsが正しくインストールされていない可能性があります。ターミナルで node -v と入力して、バージョンが表示されるか確認してください。表示されない場合は、インストーラーページから再インストールしてください。

各ツールの認証方法を確認してください。Claude Codeの場合はAnthropicアカウント（Pro以上）が必要です。Codex CLIはOpenAIのAPIキー、Gemini CLIはGoogleアカウントでの認証が必要です。

利用制限に達している可能性があります。Claude Codeなら/costで利用量を確認できます。制限に近い場合は、しばらく待つか、プランのアップグレードを検討してください。別のツールに切り替えて作業を続けるという方法もあります。

Claude Codeは変更前にバックアップを取っています。Codex CLIはサンドボックスで実行するため、承認するまで変更は反映されません。Gitで管理していれば git checkout -- ファイル名 で復元できます。

すべてのAI CLIツールは、操作中のファイル内容を各社のサーバーに送信してAI処理します。機密情報（パスワード、APIキーなど）を含むファイルを扱う際は注意が必要です。Claude Codeでは.claudeignoreに記述したファイルはAIに読み込まれません。

3つすべて日本語で問題なく使えます。Claude Codeは特に日本語の理解力が高いです。CLAUDE.mdに「日本語で返答すること」と書いておけば、常に日本語で応答してくれます。

使えます。「ターミナルを開いてコマンドを入力する」という最低限の操作だけ覚えれば、あとは日本語で指示を出すだけです。プログラミングの知識はAIが補ってくれます。

最初は1つから始めて構いません。おすすめはClaude Codeです。慣れてきたら他のツールも追加すると、AIごとの得意分野を活かした使い分けができるようになります。インストーラーを使えば3つまとめて入れることもできます。

実践事例：CSVデータから分析資料を作る

AI CLIツールの実践的な活用例として、CSVデータから市場分析ダッシュボードを自動生成するワークフローを紹介します。 「ぶんせき君」などのツールで集めたCSVデータを、Claude Codeに渡すだけで、見やすい分析資料が完成します。

どんなことができる？

📊

データのクレンジング

CSVの重複削除、欠損値の補完、フォーマット統一など、面倒な前処理をAIが自動で行います。

📈

グラフ・チャート生成

売上推移、価格分布、カテゴリ別比較など、データに適したグラフを自動で生成します。

🌐

HTMLダッシュボード化

分析結果をインタラクティブなHTMLページにまとめて、ブラウザで閲覧できる形にします。

🚀

GitHub Pagesで公開

作成したダッシュボードをそのままWeb上に公開。URLを共有するだけで誰でも閲覧可能。

作業の流れ（6ステップ）

1

CSVデータを用意する

ぶんせき君などで収集したCSVデータをデスクトップに置きます。

2

Claude Codeでデータを読み込む

「このCSVを分析して」と指示するだけで、AIがデータの構造を把握します。

3

AIがデータをクレンジング

重複の除去、価格データの数値化、カテゴリの統一など、前処理を自動実行。

4

分析・グラフ生成

データの傾向を分析し、適切なグラフやチャートを自動生成します。

5

HTMLダッシュボードを作成

分析結果をまとめたインタラクティブなHTMLページを生成します。

6

GitHub Pagesで公開

完成したダッシュボードをWeb上に公開し、URLで共有できます。
