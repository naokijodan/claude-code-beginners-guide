2

Claude Codeとは？

Claude Codeは、Anthropic社が提供するAIアシスタントツールです。 パソコンの「ターミナル」（黒い画面）から起動し、日本語で指示を出すと、 AIがファイルの作成・編集・整理・プログラミングなどを代わりにやってくれます。

何ができるの？

📄

ファイル作成・編集

「請求書のテンプレートを作って」「この文章を校正して」など、ファイル操作をAIに任せられます。

🌐

Webページ作成

「自己紹介ページを作って」と言うだけで、HTMLファイルを自動生成。プログラミング知識は不要です。

🔍

情報の検索・整理

大量のファイルの中から必要な情報を探したり、データを整理してまとめたりできます。

🛠

自動化・効率化

繰り返し作業を自動化するスクリプトを作ったり、面倒な作業をAIに肩代わりしてもらえます。

ChatGPTとの違い

多くの方が使ったことのあるChatGPTは「チャットで質問して回答をもらう」ツールです。 Claude Codeは一歩進んで、AIが直接あなたのパソコン上で作業を実行してくれます。

  • ChatGPT：「こうすればいいですよ」と教えてくれる（アドバイザー）
  • Claude Code：「やっておきましたよ」と実際にやってくれる（実行者）

つまり、回答をコピーして自分で作業する手間がなくなります。 指示を出すだけで、ファイルが作られ、コードが書かれ、結果が目の前に現れます。

📚 用語解説
AI（人工知能）
人間のように考え、判断し、作業ができるコンピュータプログラム。Claude Codeの中身はAnthropicのAI「Claude」です。
CLI（コマンドラインインターフェース）
マウスでクリックする代わりに、キーボードで文字（コマンド）を入力してパソコンを操作する方法。Claude Codeはこの方式で動きます。
ターミナル
CLIを使うためのアプリ。Macでは「ターミナル」、Windowsでは「PowerShell」や「コマンドプロンプト」と呼ばれます。黒い画面のアレです。
Anthropic（アンソロピック）
Claude Codeを開発した米国のAI企業。AIの安全性を重視していることで知られています。

Phase 3 — 使う
6

はじめの一歩

Claude Codeを起動する

ターミナルを開いて、以下のコマンドを入力します。

ターミナル 
claude

これだけです。claudeと入力してEnterキーを押すと、 Claude Codeが起動し、入力待ち状態になります。

基本的な対話の仕方

起動したら、日本語で「やりたいこと」をそのまま書くだけです。 プログラミングの知識は一切不要です。

💬 対話の例：挨拶
あなた こんにちは。何ができるか教えて。
Claude こんにちは。ファイルの作成・編集、プログラミング、データ整理、Web検索など、さまざまな作業をお手伝いできます。何から始めますか？
💬 対話の例：ファイル作成
あなた 「自己紹介.txt」というファイルを作って。名前は田中太郎、趣味は読書と散歩。
Claude 自己紹介.txtを作成しました。

内容:
名前：田中太郎
趣味：読書、散歩

終了する方法

Claude Codeを終了するには、以下のいずれかを入力します。

ターミナル 
/exit

または、キーボードで Ctrl + C を2回押しても終了できます。

覚えておきたい基本コマンド

入力 何が起きるか
claude Claude Codeを起動する
/exit Claude Codeを終了する
/help ヘルプを表示する
/clear 会話をリセットする
Ctrl + C 実行中の作業を中断する
💡 コツ

Claude Codeへの指示は、具体的であるほど良い結果が得られます。「いい感じにして」より「A4サイズの請求書テンプレートをExcel形式で作って。項目は日付、品名、数量、単価、合計」のように具体的に伝えましょう。

CLAUDE.md — AIへの作業ルール設定

プロジェクトのフォルダに CLAUDE.md というファイルを置くと、 Claude Codeが毎回そのルールを読み込んで、あなたの好みに合わせた動作をしてくれます。

📄 CLAUDE.mdの例
CLAUDE.md 
# プロジェクトルール

## 言語
- 日本語で返答すること
- コメントも日本語で書くこと

## コーディングスタイル
- ファイル名は日本語OK
- 作成したファイルは必ず報告すること

## 禁止事項
- 既存ファイルを確認なしに削除しないこと
📚 用語解説
Markdown（マークダウン）
「# 見出し」「**太字**」のように、記号で文章の書式を指定する書き方。CLAUDE.mdはこの形式で書きます。特別なソフトは不要で、メモ帳でも書けます。
Git（ギット）
ファイルの変更履歴を記録する仕組み。「いつ、何を変えたか」を記録できるので、間違えても前の状態に戻せます。
GitHub（ギットハブ）
Gitで管理したファイルをインターネット上に保存・共有するサービス。無料で使えます。

ナレッジ管理：Obsidianとの連携

Claude Codeで作業した内容を記録・整理するのに、Obsidianというノートアプリが相性抜群です。 作業ログ、学んだこと、メモなどを蓄積していくことで、自分だけのナレッジベースが作れます。

なぜObsidianなのか

  • ローカル保存：データが自分のPC内にあるので、クラウド障害やサービス終了の心配がない
  • Markdown形式：Claude Codeが出力する形式とそのまま互換性がある
  • 双方向リンク：ノート同士を繋げて、知識のネットワークが作れる
  • 無料で使える：基本機能は完全無料

Claude Code × Obsidian の活用例

1

開発ログの自動作成

Claude Codeに「今日の作業内容をObsidianノートにまとめて」と頼むと、作業ログを自動生成してくれます。

2

議事録の整理

会議のメモをClaude Codeに渡して「要点を整理してObsidianに保存して」と指示できます。

3

学習ノートの蓄積

新しく学んだことをノートに残し、後からClaude Codeに「過去のノートを参照して」と依頼できます。

よくある質問・トラブル対応

Node.jsが正しくインストールされていない可能性があります。ターミナルで node -v と入力して、バージョンが表示されるか確認してください。表示されない場合は、インストーラーページから再インストールしてください。

Anthropicのアカウントが有効か、サブスクリプション（Pro以上）に加入しているか確認してください。ブラウザで console.anthropic.com にログインして確認できます。

利用制限に達している可能性があります。/costコマンドで利用量を確認してください。制限に近い場合は、しばらく待つか、プランのアップグレードを検討してください。

Claude Codeはファイル変更前にバックアップを取っています。/undoで直前の変更を元に戻せます。また、Gitで管理していれば git checkout -- ファイル名 で復元できます。

Claude Codeは操作中のファイル内容をAnthropicのサーバーに送信してAIが処理します。機密情報（パスワード、APIキーなど）を含むファイルを扱う際は注意が必要です。.claudeignoreファイルに記述したファイルは、AIに読み込まれません。

日本語で問題なく使えます。質問も指示もすべて日本語でOKです。CLAUDE.mdに「日本語で返答すること」と書いておけば、常に日本語で応答してくれます。

使えます。ただし「ターミナルを開いてコマンドを入力する」という最低限の操作は必要です。それさえできれば、あとは日本語で「こういうものを作って」「このファイルを整理して」と伝えるだけです。プログラミングの知識はClaude Codeが補ってくれます。

サブスクリプション制なので、月額固定（Pro: $20/月）です。従量課金ではないため、使いすぎで高額請求される心配はありません。ただし、利用量に上限があり、超えるとレート制限がかかります。

⚠️ セキュリティの注意

パスワード、APIキー、個人情報などの機密データは、Claude Codeに直接入力しないでください。環境変数や.envファイルで管理し、.claudeignoreに追加して保護しましょう。
